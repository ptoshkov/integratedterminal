Shell Path:
Full path to the shell executable, such as C://Git//bin//bash.exe, or only the executable name if the executable is on the path.

Shell Arguments:
Comma-separated list of command line arguments for the shell, for example ["--login", "--interactive"].

options:
See https://xtermjs.org/docs/api/terminal/interfaces/iterminaloptions/ for a description of the fields and their values.
