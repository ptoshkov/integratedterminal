VS Code-style integrated terminal for the MATLAB IDE.

## Getting Started Guide
1. Download and install the toolbox from the MATLAB Central (LINK TBD) or the Add-On Explorer inside the MATLAB IDE.
2. Click the APPS tab on the MATLAB Toolstrip.
3. Click the dropdown arrow to show all apps.
4. Click the Integrated Terminal app. The integrated terminal will open as a docked figure.<p><img width="645" height="263" alt="image" src="https://github.com/user-attachments/assets/661ef950-f311-4ab0-80a4-6bf49ff12a06" /></p><p><img width="775" height="406" alt="image" src="https://github.com/user-attachments/assets/7ce287e3-109b-46b4-a8d1-12dbc5dc0a24" />
